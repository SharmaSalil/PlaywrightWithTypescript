# PlaywrightWithTypescript

A comprehensive API automation testing framework built with **Playwright** and **TypeScript** designed to assert and validate API responses effectively.

## 📋 Description

This project demonstrates automated API testing using Playwright with TypeScript. It provides a structured approach to validate API responses, assertions, and end-to-end testing scenarios with type safety and modern JavaScript/TypeScript practices.

## 🛠️ Tech Stack

- **Playwright**: Cross-browser automation library for end-to-end testing
- **TypeScript**: Typed superset of JavaScript for enhanced code quality and maintainability
- **API Testing**: Focused on API response validation and assertions

## ✨ Features

- Type-safe API automation using TypeScript
- Comprehensive API response assertions
- Modular test structure for easy maintenance
- Cross-browser support
- Automated testing workflows

## 📦 Installation

Clone the repository:
```bash
git clone https://github.com/SharmaSalil/PlaywrightWithTypescript.git
cd PlaywrightWithTypescript
```

Install dependencies:
```bash
npm install
```

## 🚀 Getting Started

### Running Tests

Execute all tests:
```bash
npm test
```

Run tests in headed mode:
```bash
npm run test:headed
```

Run tests in debug mode:
```bash
npm run test:debug
```

## 📝 Configuration

### TypeScript Configuration

Ensure `tsconfig.json` is properly configured for your project needs.

### Playwright Configuration

Update `playwright.config.ts` to customize:
- Browser configurations
- Timeout settings
- Reporter options
- Test parallelization
